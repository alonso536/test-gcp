# Core Api Data Provider

Backend escalable construido sobre Node.js y TypeScript para la gestión de consultas a la base de datos Core. Implementa una arquitectura modular para optimizar procesos de lectura, garantizando la integridad de los datos y la eficiencia en la respuesta mediante patrones de diseño modernos.

### Pasos para correr aplicación

1. Copiar archivo .env.template y renombrar a .env
2. Rellenar variables de entorno
3. Ejecutar npm install
4. Ejecutar npx prisma db pull
5. Ejecutar npx prisma generate
6. Ejecutar npm run dev:start para correr en modo de desarrollo
